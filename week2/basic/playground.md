# 리스트 문법 익히기
리스트는 순서가 있음
내용 수정 가능
중복값 허용

리스트 내부 개수 구할때는 len(리스트명)

리스트는 모든 데이터 타입이 들어갈 수 있음. (객체도 올 수 있나?) -> 당연히 가능

리스트의 타입은 list로 정해져있음

### list 인덱싱 관련
리스트는 인덱싱은 0부터 시작
리스트 끝부터 거꾸로 세려면 -1을 맨 끝으로 -2, -3... 으로 인덱싱

리스트 요소의 개수를 넘어서 인덱싱하면? -> 에러발생 (다시 첫 번째 요소로 되돌아오지 않음)

범위를 지정해서 인덱싱 하려면?
mylist[시작:끝-1]
인덱스 숫자 비워두면 끝까지 인덱싱
mylist[4:-1] 과 mylist[4:]는 같지않음
: 뒤에오는 값은 끝-1을 의미하기 때문에 -1을 하면 마지막 요소가 포함되지않음
 
주의
mylist[3:2] 를 한다고 거꾸로 인덱싱 하지 않음. 출력할 경우 []로 출력됨
